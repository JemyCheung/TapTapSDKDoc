## InitSDK
```
/**
 * @param appid 心动AppID
 * @param aOrientation 屏幕方向，0表示横屏，1表示竖屏
 * @param channel 渠道号
 * @param version 版本号
 * @param enableTapDB 是否开启TapDB
 */
public static void InitSDK(string appid, int aOrientation, string channel, string version, bool enableTapDB)
```

参数 | 必选 
:--- | :--- 
appid | Y
aOrientation | Y
channel | N
version | N 
enableTapDB | Y
